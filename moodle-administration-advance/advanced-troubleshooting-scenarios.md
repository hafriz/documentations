# Advanced Troubleshooting Scenarios

## Scenario 1: Users Cannot Log In

Check authentication plugin status, identity-provider availability, DNS/TLS issues, account suspension, lockout policy, session storage, and recent configuration changes.

## Scenario 2: Scheduled Tasks Are Backlogged

Review cron frequency, task locks, failed tasks, server load, long-running reports, email delivery issues, and ad-hoc task queue growth.

## Scenario 3: Quiz Performance Is Poor

Check concurrent usage, database load, cache health, PHP workers, session handling, question bank complexity, network latency, and recent plugin/theme changes.

## Scenario 4: File Uploads Fail

Review PHP upload limits, web-server limits, reverse-proxy limits, disk capacity, file permissions, antivirus scanning, and `moodledata` availability.
