# Moodle Administration Intermediate

## Course Information for Trainees

| Item | Details |
|---|---|
| **Training Title** | Moodle Administration Intermediate |
| **Training Duration** | 2 Days, 9:00 AM – 4:00 PM |
| **Learning Format** | Short explanations, guided Moodle demonstrations, hands-on labs, scenario practice, and reflection checkpoints |
| **Who Should Attend** | Moodle administrators, LMS coordinators, help desk leads, instructional technology staff, and junior system administrators |
| **Recommended Experience** | Completion of Moodle Administration Basics or equivalent day-to-day Moodle support experience |
| **Course Level** | Intermediate |
| **Version** | 1.1 |

## What You Will Learn

This course is written for trainees who already understand Moodle fundamentals and now need to operate a Moodle site with more confidence, consistency, and governance.

The intermediate course bridges the gap between the basics package and the advanced operations package. You will practice repeatable administrative workflows, safer configuration changes, meaningful reporting, light automation, practical troubleshooting, and stakeholder communication.

## How to Use This Course

1. Read the module objectives before each session.
2. Follow the demonstrations in a non-production Moodle site when possible.
3. Complete each lab using your organization’s naming, enrolment, support, and change-control rules.
4. Keep your completed checklists as job aids for future Moodle administration work.
