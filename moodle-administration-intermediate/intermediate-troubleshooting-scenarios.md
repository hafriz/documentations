# Intermediate Troubleshooting Scenarios

## Scenario 1: Teacher Cannot See a Course

Inspect category visibility, course visibility, enrolment status, role assignment context, suspended user status, and dashboard filtering.

## Scenario 2: Learners Cannot Access a Quiz

Check enrolment status, group restrictions, activity availability dates, completion restrictions, password or network restrictions, overrides, and whether the quiz is hidden.

## Scenario 3: Course Page Loads Slowly

Review recent content changes, embedded media, large files, question bank size, course format, completion tracking, reports, scheduled task backlog, and cache status before escalating to infrastructure teams.

## Scenario 4: Unexpected Permission

Trace the user's permissions from system to category, course, and activity context. Look for temporary role assignments, cohort-based enrolments, role overrides, manager roles, and plugin-specific capabilities.

## Scenario 5: Report Data Looks Incorrect

Confirm report filters, date ranges, time zones, completion criteria, cron status, activity configuration, user suspension state, and whether logs were purged according to retention settings.
